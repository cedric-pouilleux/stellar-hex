# Gas Giant with Rings — Native Three.js

Banded gas giant with a ring disc using a preset archetype, built from the
`/core` entry point.

```ts
import * as THREE from 'three'
import {
  useBody,
  DEFAULT_TILE_SIZE,
  buildBodyRings,
  RING_ARCHETYPES,
} from '@cedric-pouilleux/stellar-hex/core'

const scene = new THREE.Scene()
scene.add(new THREE.AmbientLight(0xffffff, 0.3))
const sun = new THREE.DirectionalLight(0xffffff, 2)
sun.position.set(5, 3, 3)
scene.add(sun)

const body = useBody(
  {
    type:        'gaseous',
    name:        'Jovian',
    radius:      2,
    temperatureMin: 90,
    temperatureMax: 130,
    rotationSpeed:  0.003,
    axialTilt:      0.05,
  },
  DEFAULT_TILE_SIZE,
)
scene.add(body.group)

// Attach a Saturn-like ring disc
const rings = buildBodyRings({
  group:        body.group,
  radius:       body.config.radius,
  rotationSpeed: body.config.rotationSpeed,
  variation:    body.variation.rings!, // rings are auto-generated by useBody
})
```

The full list of available ring archetypes is documented on
[`RING_ARCHETYPES`](/api/) in the API reference.
